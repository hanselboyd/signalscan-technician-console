# SignalScan v1.0.0 Launch Instructions

1. Unzip the SignalScan release package to a local folder.
2. Run `SignalScan.TechnicianConsole.exe` from the unzipped folder.
3. Run the app first on a non-client machine or VM.
4. Click `Run Read-Only Scan` and confirm the scan completes without making system changes.
5. Export a PDF report and review it before sharing.
6. Remember: SignalScan v1.0.0 does not repair, clean up, delete files, or change system settings.
