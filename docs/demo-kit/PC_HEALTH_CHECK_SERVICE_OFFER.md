# PC Health Check by 909 Signal IT

Powered by SignalScan

## What It Is

PC Health Check by 909 Signal IT is a technician-reviewed Windows computer assessment for residential and small business clients. SignalScan collects read-only diagnostic information, organizes the findings, and helps produce a clear report that explains the device's current health in practical language.

## Who It Is For

- Home users who want a plain-English review of a Windows PC before problems become urgent.
- Small business owners who rely on workstations for daily operations.
- Clients who are unsure whether slowness, storage pressure, security warnings, or update issues need professional attention.
- New support clients who need a baseline assessment before recommending next steps.

## What the Customer Receives

- A technician-reviewed PC health report.
- A summary of system profile, performance, maintenance, and security posture findings.
- Clear status language: Good, Attention Needed, Critical, or Review Required.
- Practical next-step recommendations reviewed by a technician.
- Optional Markdown draft/debug output for internal technician review.

## What SignalScan Checks

- Windows edition, version, build, uptime, CPU, RAM, fixed-drive storage, manufacturer/model, and BIOS/firmware details.
- Performance indicators such as CPU usage snapshot, memory usage, disk free space, startup entry count, visible process count, and uptime.
- Maintenance indicators such as pending reboot signals, Windows Update history timestamps, recent warning/error event counts, and disk health where available.
- Security posture indicators such as Windows Defender status, firewall profile status, BitLocker status, local administrator count, and Windows support status where available.

## What SignalScan Does Not Do

- It does not repair, clean up, delete files, optimize, uninstall, disable, or change system configuration.
- It does not write registry keys, modify services, change drivers, alter Defender/firewall/BitLocker settings, manage users, change group policy, trigger Windows Update, or modify startup items.
- It does not send diagnostics to the cloud, use telemetry, sync data, or call external AI services.
- It does not replace hands-on technician judgment, hardware testing, vendor tools, or follow-up service when needed.

## Suggested Introductory Price Range

Suggested introductory range: **$49 to $99 per PC**, depending on location, appointment format, and whether the review is remote, onsite, or bundled with another service.

For small businesses with multiple workstations, consider a per-device discount or a fixed assessment block.

## Recommended Follow-Up Service Packages

- **Basic Tune-Up Review**: Review startup entries, common performance issues, storage pressure, update status, and practical maintenance next steps.
- **Security Review**: Review Defender/antivirus posture, firewall status, local administrator exposure, BitLocker status, and Windows lifecycle concerns.
- **Storage & Performance Review**: Review disk capacity, RAM pressure, process count, uptime, drive health indicators, and performance findings.
- **Business Workstation Review**: Review device baseline, reliability risks, security posture, update status, and support recommendations for small business workstations.

## Read-Only Disclaimer

SignalScan v1.0.0 is read-only. It creates a technician-reviewed report from available diagnostic information, but it does not repair, clean up, delete files, or change system settings. Any follow-up service or repair work must be performed separately by a technician with the client's approval.
