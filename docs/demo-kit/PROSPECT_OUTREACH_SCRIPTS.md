# Prospect Outreach Scripts

These scripts are sample copy for 909 Signal IT. Adjust names, timing, and pricing before sending. Avoid using real client details in public examples.

## Text Message to a Local Residential Customer

Hi [First Name], this is 909 Signal IT. I am offering a PC Health Check powered by SignalScan for local Windows PCs. It creates a technician-reviewed report covering system health, performance, maintenance, and security posture. SignalScan is read-only, so it does not repair, delete, clean up, or change settings during the scan. Would you like me to check your PC and review the report with you?

## Email to a Small Business Owner

If helpful, I can run the check on one workstation first and walk you through the report.

Thank you,

909 Signal IT
